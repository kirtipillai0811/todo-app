# To-Do App

A simple and modern To-Do List web application built using HTML, CSS, and JavaScript.

## 🚀 Features
- Add tasks
- Edit tasks (✏️ button)
- Delete tasks (🗑 button)
- Mark tasks as completed
- Saves tasks using localStorage

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
